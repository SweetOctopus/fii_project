Language relatedness

The dendrogram displayed on this page was created using the online hierarchical clustering tool by Revoledu.

It employs a hierarchical agglomerative clustering algorithm applied to the similarity matrix, with clusters grouped using single linkage clustering.

The matrix comprises 11 Arabic varieties and 11 dimensions corresponding to the usages of fii investigated in this study. The values associated with each dimension range from 0% to 100%, reflecting the judgments expressed by the native consultants. These values are the same as those represented in the maps and can be calculated from the database as described on the Geographic Distribution page.

A noteworthy feature of the dendrogram is that the resulting structure does not solely reflect groupings based on the geographical distribution of the investigated varieties. This suggests that morphosyntactic change is driven by internal language factors that are independent of language contact. For contrasting results based exclusively on lexical data, see Salameh, Bouamor, & Habash (2018).
